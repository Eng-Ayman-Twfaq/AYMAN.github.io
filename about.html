من أنا؟

أنا أيمن توفيق عبدالودود من اليمن 🇾🇪، مبرمج ومهندس تقني محترف متخصص في تطوير الحلول البرمجية المتكاملة.

أمتلك خبرة في عدة مجالات تقنية منها:

  • تطوير المواقع باستخدام Laravel و PHP
  • بناء تطبيقات الموبايل باستخدام Flutter
  • برمجة الأنظمة الذكية والتحكم باستخدام Arduino
  • تحليل وتصميم قواعد البيانات بكفاءة
  • إدارة المشاريع التقنية ومشاركات مفتوحة المصدر
  • تطوير واجهات المستخدم باستخدام React.js
  • حلول IoT والأنظمة المدمجة

رحلتي المهنية

2019 - البداية

كانت انطلاقتي في عالم البرمجة من خلال تعلم لغة C++ وبناء الأساسيات المنطقية والخوارزميات البرمجية.

2020 - قواعد البيانات

تعلمت تحليل وتصميم قواعد البيانات، ونمذجة البيانات باستخدام ERD وربطها بالتطبيقات عبر SQL و MySQL.

2021 - C#

التعمق في برمجة تطبيقات سطح المكتب باستخدام C# و .NET Framework وإنشاء واجهات مستخدم متقدمة.

2022 - Arduino

برمجة المتحكمات وربط الأنظمة الذكية باستخدام الحساسات وتقنيات IoT وإنشاء مشاريع عملية.

2023 - Laravel

بناء مواقع وتطبيقات ويب احترافية باستخدام Laravel و MySQL وإنشاء واجهات برمجة API متكاملة.

2024 - Flutter

تطوير تطبيقات Android و iOS متعددة الواجهات والوظائف باستخدام Flutter وإنشاء تطبيقات كاملة.
